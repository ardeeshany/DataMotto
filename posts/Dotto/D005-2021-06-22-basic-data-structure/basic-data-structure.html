Factor is a data structure used to represent categorical data.
For example, a data field like gender will contain only values of male,female or other. Factors in R are stored as a vector of integer that each integer has a label.
One of the most important uses of factors is in statistical modeling; since categorical variables enter into statistical models differently than continuous variables, storing data as factors insures that the modeling functions will treat such data correctly.
for more information,visit here.

Tuple is an ordered collection of objects in Python.
You can’t change tuple atfer defining it(unlike list).

A list is a collection of elements which can be of different types.
A vector having all elements of the same type is called atomic vector(vector) but a vector having elements of diffrent types is called list.
We can also set tags(a,b,c are tags here) for list elements.Doing this, our list will be like dictionary in Python.
as.list() is used to convert an object to list. These objects can be Vectors, Matrices, Factors, and data frames.
You can also see this and this for more information.

Python’s dictionaries allow you to connect pieces of related information.Each piece of information in a dictionary is stored as a key-value pair.
When you provide a key, Python returns the value associated with that key.
You can convert list to dict using list index as the keys.
Please check here for details.

Data Frame is a special case of a list where all elements are the same length.The main usage of data frame is storing tabular data.
Data Frames can be created using data.frame().
as.data.frame() function in R Language is used to convert an object to data frame. These objects can be Vectors, Lists, Matrices, and Factors.
you can also see here

A DataFrame is a table. It contains an array of individual entries, each of which has a certain value. Each entry corresponds to a row (or record) and a column. Data frame is one of the most commonly used pandas objects.
df.head(n) returns first n rows of data frame.
df.tail(n) returns last n rows from data frame.
pandas.DataFrame(list) converts the list to data frame.
as.matrix(df) converts dataframe to a numeric matrix.
you can also convert dictionary to dataframe using pandas.DataFrame.from_dict(data, orient = { 'columns','index'} , dtype=None, columns=None).
Please visit here for details.
