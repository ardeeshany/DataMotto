R.Factor is a data structure used to represent categorical data.For example, a data field like gender will contain only values of male,female or other. Factors in R are stored as a vector of integer that each integer has a label. One of the most important uses of factors is in statistical modeling; since categorical variables enter into statistical models differently than continuous variables, storing data as factors insures that the modeling functions will treat such data correctly. for more information,see https://www.datamentor.io/r-programming/factor/

Python. Tuple is an ordered collection of objects in Python. You can’t change tuple atfer defining it(unlike list).

 Codes:


#create a factor using factor():
factor1 <- factor(c("male","female","female","male"))
levels(factor1) #shows levels of factor1
table(factor1) #returns number of each level
#we can determine the order of levels:
factor2 <- factor(c("pass","pass","fail","pass","fail"), levels = c("pass" , "fail"))
factor2
factor2[c(3,4)] #access 3rd and 4th element of factor2
#illustrate how to modify elements(CANNOT assign values outside levels):
factor1[1] <- "female"
levels(factor1) <- c(levels(factor1),"other") #adding new level
factor1[2] <- "other" #modify 2nd element of factor1
factor1

Python. Python’s dictionaries allow you to connect pieces of related information.Each piece of information in a dictionary is stored as a key-value pair. When you provide a key, Python returns the value associated with that key.

 Codes:

#We can also set tags(a,b,c are tags here) for list elements.Doing this,
#our list will be like dictionary in Python:

list2
list2[[1]] #access 1st element of list2
list2$b #access element of list2 named b
list2$b <- "bye!" #modify 2nd element of list2(element named "b")
list2$d <- 10.5 #add new element to list
list2
list2$c <- NULL #delete element named c from list2
list2
dict1 = {"a":1,"b":2,"c":3} #define a dictionary
print("a is a key and it's value is :",dict1.get("a"))
print("True") if "b" in dict1 else print("False")#search in dictionary
dict1["d"] = 4 #add a new pair to dictionary
dict1["c"] = "10" #modify c's value
del dict1["a"]
print("dict1 after deleting a_1 pair :",dict1)
print("length of dict1 :",len(dict1))
#Looping through all key-value pairs:
for alphabet,number in dict1.items():
  print(alphabet+" : ",number)
#looping through all the keys:
for alphabet in dict1.keys():
  print(alphabet)
#looping through all values:
for number in dict1.keys():
  print(number)

R. Data Frame is a special case of a list where all elements are the same length.The main usage of data frame is storing tabular data. Data Frames can be created using data.frame(). you can also see https://www.datamentor.io/r-programming/data-frame/

Puthon. A DataFrame is a table. It contains an array of individual entries, each of which has a certain value. Each entry corresponds to a row (or record) and a column. Data frame is one of the most commonly used pandas objects.
